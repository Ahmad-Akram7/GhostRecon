// 02 — SCAN TYPES

12 modes. Every scenario.

From silent recon to full-spectrum aggressive scans — pick the right tool for the job.

PING SWEEP
-sn
ICMP echo check — confirm host is alive before investing in deeper scans.
RECON
SYN SCAN
-sS
Half-open TCP handshake. Fast, stealthy, avoids full connection logging.
STEALTH
UDP SCAN
-sU
Discover open UDP services often missed by TCP-only scans.
RECON
VERSION DETECTION
-sV
Fingerprint exact service versions running on open ports.
ENUM
OS DETECTION
-O
Remote OS fingerprinting via TCP/IP stack analysis.
ENUM
AGGRESSIVE
-A

AGGRESSIVE
STEALTH
-sS -Pn
SYN scan with no ping — evades firewalls that block ICMP.
STEALTH
FULL TCP
-sT
Full TCP connect scan — works without root, more reliable in some environments.
RECON
SLOW SCAN
-sS -T2 -Pn
Low-speed scan to evade IDS/IPS rate-based detection systems.
EVASION
NO PING
-Pn
Skip host discovery — scan even hosts that block ping probes.
STEALTH
TRACEROUTE
--traceroute
Map the network path to target — identify hops and routing topology.
RECON
TOP PORTS
--top-ports 100
Scan the 100 most common ports — fast coverage of likely attack surface.

// 03 — INSTALLATION

Up and running in 60 seconds.

No package hell. No venv. Just Python and Nmap.

01

Install Nmap

# Debian / Ubuntu sudo apt update && sudo apt install nmap # macOS brew install nmap # Windows — download from nmap.org/download.html
02

Clone the repo

git clone https://github.com/Ahmad-Akram7/GhostRecon.git cd GhostRecon
03

Run the scanner

# Basic scanner python GhostRecon.py # Advanced interactive scanner (recommended) python nmapscanner.py
04

View your results

ls results/ # → scan_20250614_154201.txt cat results/scan_20250614_154201.txt
